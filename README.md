# Sobre
RPG de terminal desenvolvido em Swift, utilizando todas as tecnologias apresentadas durante um workshop realizado em parceria com a Apple.

# Corpo do projeto
A estrutura do código foi construída com structs representando as “classes” do jogo, além do uso de estruturas condicionais, loops, arrays e enums para organizar a lógica e o fluxo da aplicação.

# Proposta
O objetivo do projeto é identificar e avaliar a qualidade do conhecimento adquirido ao longo do workshop, reunindo todas as lógicas aprendidas e aplicando-as em um desafio final.

# Conclusão
Ao finalizar este projeto, consolidei meu entendimento sobre a sintaxe do Swift e suas estruturas básicas. Isso fortaleceu minha base na linguagem e facilitará meu avanço futuro, especialmente em áreas mais complexas, como o desenvolvimento de aplicações completas.
